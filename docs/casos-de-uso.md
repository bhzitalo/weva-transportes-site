# 🎯 Casos de Uso – Weva Transportes

## 1. Introdução

Este documento descreve os principais casos de uso do sistema da Weva Transportes.

Os casos de uso representam as interações entre os usuários e o sistema, demonstrando como as funcionalidades serão utilizadas na prática.


## 2. Atores do Sistema

### Usuário  
Pessoa que acessa o site em busca de informações ou serviços.

### Administrador (Empresa)  
Responsável por responder contatos e manter as informações atualizadas.

## 3. Lista de Casos de Uso

- UC01 – Visualizar informações da empresa  
- UC02 – Visualizar serviços  
- UC03 – Visualizar frota  
- UC04 – Entrar em contato  
- UC05 – Solicitar orçamento  
- UC06 – Acessar WhatsApp  


## 4. Detalhamento dos Casos de Uso


### UC01 – Visualizar informações da empresa

**Ator:** Usuário  

**Descrição:**  
O usuário acessa o site e visualiza informações institucionais da empresa.

**Fluxo Principal:**
1. O usuário acessa a página inicial  
2. O sistema exibe informações da empresa  
3. O usuário lê o conteúdo  

### UC02 – Visualizar serviços

**Ator:** Usuário  

**Descrição:**  
O usuário acessa a página de serviços e visualiza os serviços oferecidos.

**Fluxo Principal:**
1. O usuário acessa o menu  
2. Clica em “Serviços”  
3. O sistema exibe a lista de serviços  

### UC03 – Visualizar frota

**Ator:** Usuário  

**Descrição:**  
O usuário acessa a página de frota e visualiza os veículos disponíveis.

**Fluxo Principal:**
1. O usuário acessa o menu  
2. Clica em “Frota”  
3. O sistema exibe os veículos  

### UC04 – Entrar em contato

**Ator:** Usuário  

**Descrição:**  
O usuário acessa a página de contato e visualiza as formas de comunicação.

**Fluxo Principal:**
1. O usuário acessa o menu  
2. Clica em “Contato”  
3. O sistema exibe telefone, e-mail e WhatsApp  

### UC05 – Solicitar orçamento

**Ator:** Usuário  

**Descrição:**  
O usuário preenche um formulário para solicitar um orçamento.

**Fluxo Principal:**
1. O usuário acessa a página de contato  
2. Preenche o formulário  
3. Envia a solicitação  
4. O sistema registra a solicitação  

### UC06 – Acessar WhatsApp

**Ator:** Usuário  

**Descrição:**  
O usuário entra em contato diretamente com a empresa via WhatsApp.

**Fluxo Principal:**
1. O usuário clica no botão de WhatsApp  
2. O sistema redireciona para o WhatsApp  
3. O usuário inicia a conversa  

## 5. Considerações Finais

Os casos de uso apresentados descrevem as principais interações do sistema, servindo como base para o desenvolvimento e validação das funcionalidades.
