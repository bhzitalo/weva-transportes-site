# 🧩 Wireframe – Weva Transportes

## 1. Introdução

Este documento apresenta a estrutura visual (wireframe) das páginas do site da Weva Transportes.

O objetivo é definir a organização dos elementos na interface antes do desenvolvimento, facilitando a implementação e o trabalho em equipe.


## 2. Wireframe – Página Inicial (Home)

```
[ HEADER ]
Logo | Menu (Início | Serviços | Frota | Sobre | Contato)

[ BANNER ]
Imagem principal
Título: "Transporte com segurança e conforto"
Botão: "Solicitar orçamento"

[ SERVIÇOS ]
Card 1 – Transporte de funcionários
Card 2 – Turismo pedagógico
Card 3 – Turismo religioso
Card 4 – Eventos e empresas

[ FROTA ]
Imagem + descrição:
- Vans
- Ônibus
- Micro-ônibus
- Carros

[ DIFERENCIAIS ]
- Segurança
- Conforto
- Pontualidade
- Qualidade

[ CONTATO RÁPIDO ]
Telefone | WhatsApp | Botão de contato

[ FOOTER ]
Endereço | E-mail | Redes sociais
```

## 7. Considerações Finais
